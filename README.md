# Frontend Future 2026 – Landing Page

Одностраничный лендинг для конференции **Frontend Future 2026**, посвящённой современному фронтенду.  
Сайт содержит программу, список спикеров, блоки социального доказательства, тарифы и форму регистрации.

## Особенности

- Адаптивный дизайн (mobile-first, поддержка всех размеров экрана)
- Плавная прокрутка к секциям
- Аккордеон с программой по дням
- Модальные окна с детальной информацией о спикерах
- Форма регистрации с валидацией и подтверждением
- Кастомный скроллбар в цветах бренда
- Мобильное меню-бургер
- Фиксированная шапка с эффектом размытия (glassmorphism)
- Кнопка «Наверх» при скролле
- Использование CSS-переменных для лёгкой смены темы

## Технологии

- HTML5, CSS3 (Flexbox, Grid, анимации)
- [Bootstrap 5.3](https://getbootstrap.com/) – базовая сетка и утилиты
- Vanilla JavaScript (без зависимостей)
- Иконки социальных сетей – встроенные SVG
- Аватары спикеров генерируются через [ui-avatars.com](https://ui-avatars.com/)

##  Кастомизация

Все основные цвета и шрифты заданы через CSS-переменные в блоке :root.
Чтобы изменить тему, достаточно поменять значения переменных:

css
:root {
  --accent-mid: #e11d48;   /* Основной акцентный цвет */
  --gradient-main: ...;    /* Градиент кнопок */
  --font-heading: 'Inter', ...;
}
Данные спикеров находятся в массиве speakersData внутри <script> – их легко дополнить или изменить.
